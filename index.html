Map Router Web Application

Map Router is a transportation management web application built for a church in Bossier City, Louisiana. It allows staff coordinators to manage rider contacts, build weekly pickup routes, automatically optimize stop order for the shortest path, and share a mobile-friendly driver view with turn-by-turn Google Maps integration. The app includes role-based admin access, a completed route history log, and was deployed to Microsoft Azure.

  • React 18
  • Vite
  • Tailwind CSS
  • Node.js
  • Express
  • Prisma ORM
  • Azure SQL
  • Azure App Service
  • JWT Authentication

Stormy For Shreveport Website

This high-performance landing page was strategically developed to establish an authoritative digital presence for Stormy Gage-Watts' mayoral campaign in Shreveport. The site serves as a secure hub for voter outreach, featuring an integrated PCI-compliant donation system for legal fundraising and a native CRM to capture supporter data. Executed on an expedited timeline, the project moved from a start on January 9, 2026, through a draft review on January 11, to a successful public launch on January 14.
